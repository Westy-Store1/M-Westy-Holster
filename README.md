─── M-WESTY HOLSTER ───

Uma solução de posicionamento visual e coldres de armas minimalista e intuitiva para FiveM, desenvolvida com foco em alta performance e refinamento visual.

<img width="1672" height="941" alt="m-westy-holster" src="https://github.com/user-attachments/assets/429c9f1d-963d-4197-bb79-81a6a67e1e6b" />


✦ Especificações
Compatibilidade · Suporte nativo para vRP, QBCore, ESX e modo standalone.

Performance · Estrutura leve e otimizada (baixíssimo consumo no resmon).

Instalação · Configuração simplificada e adaptável.

Dependência · Requer a biblioteca **[ox_lib](https://github.com/overextended/ox_lib)** instalada e ativa no servidor.

✦ Funcionalidades
Customização · Sistema de posicionamento (costas, cintura, peito, perna) totalmente personalizável pelo jogador.
Persistência · Memoriza a preferência do jogador por KVP nativo do cliente.
Gerenciamento · Ajustes através do arquivo de configuração centralizado.

🔒 Distribuição & Proteção
Este recurso é gratuito. Para salvaguardar os direitos autorais do projeto e evitar a cópia não autorizada da interface ou revendas fraudulentas, a estrutura lógica (client e server) encontra-se criptografada.

Apenas o arquivo de configurações (config.lua), manifesto (fxmanifest.lua) e o arquivo de licenciamento (README-NÃO-EXCLUA.md) permanecem abertos, garantindo total autonomia para você configurar os offsets e frameworks do seu servidor.
