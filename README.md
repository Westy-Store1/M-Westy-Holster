# ─── M-WESTY HOLSTER ───

Uma solução de posicionamento visual e coldres de armas minimalista e intuitiva para FiveM, desenvolvida com foco em alta performance e refinamento visual.

<img width="1672" height="941" alt="m-westy-holster" src="https://github.com/user-attachments/assets/6dd051c2-3f4d-41c5-892e-52086720fd00" />


---

### Especificações

* **Compatibilidade** · Suporte nativo para vRP, QBCore e ESX.
* **Dependência** · Requer a biblioteca [ox_lib](https://github.com/overextended/ox_lib) ativa no servidor.
* **Performance** · Estrutura leve e otimizada (baixo consumo no resmon).
* **Instalação** · Configuração simplificada e adaptável.

### Funcionalidades

* **Customização** · Permite ao jogador escolher a posição das armas (peito, costas, perna, lombar).
* **Persistência** · Salva automaticamente a posição escolhida pelo jogador por KVP.
* **Gerenciamento** · Ajustes através do arquivo de configuração.

### Distribuição & Proteção

Este recurso é gratuito. Para salvaguardar os direitos autorais do projeto e evitar a cópia não autorizada da interface ou revendas fraudulentas, a estrutura lógica (`client` e `server`) encontra-se criptografada.

Apenas o arquivo de configurações (`config.lua`), manifesto (`fxmanifest.lua`) e licenciamento (`README-NÃO-EXCLUA.md`) permanecem abertos, garantindo total autonomia para você configurar os offsets e frameworks do seu servidor.
