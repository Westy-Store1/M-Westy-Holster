# M-Westy Holster -- QBOX, QBCore, ESX, vRP Creative e Standalone.

Sistema avançado de posicionamento visual de armas no corpo para servidores de **FiveM**. Esta versão foi totalmente otimizada para oferecer o máximo de desempenho com o menor impacto de processamento possível (Resmon constante em `0.00ms`).

---

## ✨ Funcionalidades
* **Posicionamento Customizado**: Permite configurar o local de exibição das armas (pistolas, submetralhadoras, fuzis e armas brancas) anexadas ao ped do jogador.
* **Compatibilidade Multiframe**: Suporte nativo para **Qbox**, **QBCore**, **ESX**, **vRP Creative** e modo **Standalone**.
* **Integração Inteligente**: Varredura direta dos inventários do jogador ao invés de checagens nativas constantes.
* **Menu Interativo**: Interface baseada no `ox_lib` para configurar posições facilmente no jogo (Tecla padrão: **F10**).
* **Ultra Otimizado**: Estrutura reescrita usando cache ativo e mapeamento de hashes para evitar gargalos de processamento no cliente.
